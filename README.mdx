# Nuxt Website

A Nuxt 3 website with MDX support for interactive content.

## Setup

Install dependencies:

```bash
pnpm install
```

## Development

Start the development server:

```bash
pnpm dev
```

## Production

Build for production:

```bash
pnpm build
```

## Features

- ✅ Nuxt 3
- ✅ MDX support
- ✅ Content management
- ✅ Syntax highlighting
