# Nuxt Website

A Nuxt 3 website with Markdown support for content management.

## Setup

Install dependencies:

```bash
pnpm install
```

## Development

Start the development server:

```bash
pnpm dev
```

## Production

Build for production:

```bash
pnpm build
```

## Features

- ✅ Nuxt 3
- ✅ Markdown support with HTML
- ✅ Content management
- ✅ Syntax highlighting
- ✅ View transitions
